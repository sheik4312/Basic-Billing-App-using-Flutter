# Basic Billing Application
Streamline your billing with our intuitive Flutter app powered by Firebase. Manage invoices effortlessly, track payments in real-time, and elevate your business efficiency with seamless cloud integration.

## Features
 profit calculator
 Invoice checking
 Invoice printing
 Gst utilized bills
 use voice chat to search easily
